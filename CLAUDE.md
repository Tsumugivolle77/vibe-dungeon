# 简介
一款仿照《元气骑士》的roguelike游戏，采用godot引擎，可以在windows或macos上跨平台运行。

# 数值
数值设计参考元气骑士。武器有近战和远程两类。近战类可以消除弹幕，远程类有手枪（初始武器）、冲锋枪、法杖等。请发挥想象力设计出30种左右的武器。

# 操作方式
角色通过：
- WASD进行方向移动；
- 鼠标进行瞄准；
- 滚轮进行武器切换，每次切换有0.1s的后摇；
- 空格键激活角色技能。

目前只有骑士一个角色，技能是10s内双倍火力。

# 关卡设计
目前只有第一大关：森林，包含五个小关，每个小关的房间随机从房间模板中生成。设计10种左右的敌人，包括基础怪物、精英怪。怪物可以头目化，也就是加强其子弹以及体型。最终boss是曼陀罗花。

# 音乐和音效
采用免费素材库中的素材进行音乐和音效设计。
