# 优化武器系统（下一轮：逐武器差异化）

状态系统、岩浆坑（LavaPool，支持敌方/我方两种模式）、召唤友军（AllyMinion）等底层已就绪，可直接使用。
根据武器效果，结合全局数据，适当修改耗蓝量。

- 盾击：0.8s可释放一次。向前位移一段，攻击后在主角周围生成一个蓝色半透明护盾，持续0.5s，可抵抗30点伤害。美化盾牌外形（参考圣骑士盾牌）
- 霰弹枪：改为M1/M2/M3三种变种。M1=当前实现；M2弹射+更大子弹；M3射出9颗围成环状的子弹，前进中向四周扩散
- 火箭炮：子弹放大2倍，设计更符合火箭外观的子弹
- 加农炮：子弹放大2倍，设计更符合炮弹外观的子弹
- 火焰法杖：子弹前进时每0.1s有20%留下火坑（持续1s，对其上单位造成dot）。设计更符合火焰特征的子弹并放大2倍
- 冰霜法杖：子弹30%冰冻、50%冻伤，可弹射。设计冰锥形子弹并放大2倍
- 火焰喷射器：在角色前方一定距离喷出火焰扇形攻击，造成烧伤dot
- 回旋镖：向前飞出并旋转，造成帧伤、清除轨迹弹幕，大小变为3倍
- 等离子枪：有间隔地射出3发蓝色子弹（3倍大小，可弹射两次），50%麻痹
- 轨道炮：发出闪电状激光，可摧毁该方向上的导弹
- 神圣法杖：锁定地图最近3个单位，天降圣光（圆形范围伤害，50%附加神圣dot）
- 星灭者：发射3倍大小的星形子弹
- 虚空炮：爆炸处产生黑洞，吸引周围怪物并造成黑暗dot（对boss吸引力1/3）
- 雷鸣弓：2倍大小闪电箭，飞行中对周围怪物产生电弧（电系dot）；命中后召唤天雷造成圆形范围伤害并100%麻痹
- 冰霜疾风：召唤1个追踪的冰系龙卷风（范围伤害，30%冰冻/50%冻伤），清除路径弹幕，最多存在5s，击中敌人即消失
- 龙牙：挥动后向前召唤三个龙形追踪弹幕（最多5s），清除轨迹弹幕
- 圣剑：挥动后向前召唤圆弧形大型黄色剑气，清除轨迹弹幕，持续5s，攻击并击退敌人、命中不消失，50%神圣dot
